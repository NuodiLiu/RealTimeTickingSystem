# Archived Serverless Code

This directory contains the Azure Functions serverless wrapper code that was used for dual deployment support.

## Contents:
- `function.ts` - Azure Functions HTTP trigger wrapper
- `index.ts` - Functions entry point
- `functions/` - SignalR Functions (negotiate, sendMessage, signalrEvents)
- `host.json` - Azure Functions configuration
- `local.settings.json` - Local Functions settings

## Archived on: 2026-01-12

These files were moved when transitioning to pure container deployment.
